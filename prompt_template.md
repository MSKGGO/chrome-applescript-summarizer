# 한국어 뉴스 요약 프롬프트 (커스터마이즈 가능)

`summarize.py`가 외부 파일로 이걸 읽도록 설정 가능 (또는 그대로 내장). 본인 취향에 맞게 수정.

## 출력 형식

```
**기사제목 (한국어 의역)**

* **핵심 포인트1 제목:** 1~3문장 분석 (시장/경제/지정학적 함의 해석). 수치는 굵게.
* **핵심 포인트2 제목:** 1~3문장 분석.
* **핵심 포인트3 제목:** 1~3문장 분석.
* (사안이 크면 4~5번째 포인트까지 추가)

(언론사명, YYYY-MM-DD)
[원문](URL)
```

## 규칙

- 단순 사실 나열이 아니라 **시장/경제/지정학적 함의를 해석**
- 핵심 포인트 보통 3개, 사안 크면 4~5개
- 수치(%, $, 배수)는 본문에서도 굵게 (`**...**`)
- 한국어 우선, 고유명사는 원어 병기 가능
- 발행일은 본문/URL에서 추출, 미상이면 `(언론사명)`만

## 예시 출력

```
**우크라이나, 4월 두 번째 러시아 비료 공장 타격 — 이란 전쟁발 가격 상승 차단**

* **원자재 수출 인프라 표적 확장:** 우크라이나 드론이 PhosAgro PJSC 산하 Apatit JSC 복합단지를 공격. 정유시설 위주이던 후방 공격이 비료·원자재 수출 인프라로 확장됐다는 신호.

* **호르무즈 봉쇄 + 수·중 수출제한 → 질소비료 +100%:** 이란 전쟁으로 호르무즈 사실상 폐쇄, 글로벌 해상 비료 교역의 약 1/3 차단 → 질소 비료 가격이 **전쟁 이전 대비 2배** 급등. 농업·식품 인플레 압력 재가속 위험.

* **이중용도 시설 — 비료 ≈ 폭약 원료:** 질소비료 공정이 질산·폭발물 공정과 기술적으로 연결. 경제 타격을 넘어 러시아 탄약 생산능력에도 압박을 가하려는 다목적 포석.

(Bloomberg News, 2026-04-26)
[원문](https://www.bloomberg.com/...)
```
